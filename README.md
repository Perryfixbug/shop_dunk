
# Shopdunk
Đây là 1 website lấy cảm hứng từ website shopdunk.com



## Tech Stack

**Client:** Next, TailwindCSS, Shadcn

**Server:** Node, Express


## Installation

Trong thư mục /client
```bash
    npm init
    npm install
    npm run dev
```
Trong thư mục /server
```bash
    npm init
    npm install
    npm run dev
```
## Deployment

Chạy câu lệnh sau ở mỗi thư mục /client /server

```bash
  npm run dev
```
